### ⚙️ Installation & Usage
To run any of these projects locally:
1. Clone the repo:
   ```bash
   git clone [https://github.com/KinSlay3rS/Machine-Learning-Projects.git](https://github.com/KinSlay3rS/Machine-Learning-Projects.git)
	```
2. Create python virtual environment using python venv or conda.
3. After creating the env and running install the requirments using:
	```pip install -r requirements.txt```
4. To view the .ipynb file use jupyter. Inside shell, Write : ```jupyter notebook```
5. To run the app.py, Write ```streamlit run app.py```
