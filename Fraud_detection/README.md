# Financial Fraud Detection System

## Project Overview
This project is a machine learning-based application designed to detect fraudulent financial transactions. It includes a comprehensive data analysis pipeline and a real-time prediction interface built with Streamlit. The model analyzes transaction patterns, such as transfer types and balance changes, to identify suspicious activity.

## Features
- **Exploratory Data Analysis (EDA):** Detailed visualization of transaction trends and fraud distribution.
- **Machine Learning Pipeline:** A robust preprocessing and classification pipeline using Scikit-Learn.
- **Web Interface:** A user-friendly Streamlit dashboard for manual transaction verification.
